## Codelabs

A [Codepen](codepen.io)-integrated classroom web application for programming
course instructors to teach web design (HTML, CSS, Javascript).
Students can write and edit HTML, CSS, and Javascript code from within
the website, submit their code to the classroom, see and favorite work
from their classmates, and ping other classmates and/or the instructor(s)
for help from within the website.

Built using Ruby on Rails.
